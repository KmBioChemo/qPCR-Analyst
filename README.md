## Features

- ✅ Standard curve validation (R², efficiency, slope)
- ✅ ΔΔCt expression analysis
- ✅ Welch t-test + ANOVA + Benjamini-Hochberg FDR correction
- ✅ Housekeeping gene stability (geNorm M-value)
- ✅ Automatic outlier detection
- ✅ QC report with pipetting precision heatmap
- ✅ Publication-ready PNG export
- ✅ Excel export of all results
- ✅ Save/Load sessions
- ✅ Works offline — single HTML file, no install
